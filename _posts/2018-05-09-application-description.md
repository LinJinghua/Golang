# [猫眼电影](http://maoyan.com) - 订票业务描述

------

**小组信息见[Dashboard](https://github.com/system-analysis-and-design/dashboard/blob/master/maoyan-modeling-exercises.md)**。

## 概述
猫眼，不只是电影。还是集影视娱乐信息、在线购票、影评剧评等服务的一站式泛娱乐平台，不论电影电视还是综艺，都可以找到你喜欢看的内容，看片追剧和粉爱豆变得方便。

主要功能
 - 全方位娱乐信息流，随时掌握娱乐动态
 - 影视综艺资讯、热门短视频、明星专访、娱乐八卦、秒级票房和收视率，快速了解娱乐圈动态
 - 真实购票评分和专业评分，多重保险拒绝烂片
 - 权威的专业影评人、真实的购票用户、资深的影迷作者，全民帮你选好片
 - 百万级大娱乐资料库，电影电视综艺和明星的百科全书
 - 新预告、高清剧照、幕后揭秘和明星花絮，尽在猫眼
 - 提前在线选座，不排队轻松霸好位
 - 覆盖全国7000+家影院，实时查询放映场次，提前选好座不排队
 - 正版电影周边，超多爱豆福利
 - 丰富精美的正版电影周边火热抢购，更有明星见面会等爱豆福利

本文只介绍其中的订票功能。
